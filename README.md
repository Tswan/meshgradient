# Mesh Gradient
Use WebGL radial gradients to create Mesh Gradient effects

![Mesh Gradient Banner](assets/Banner.gif)

# Installation
## Framer
[Framer Plugin](https://www.framer.com/marketplace/plugins/mesh-gradient-maker/)

[Framer Component](https://framer.com/m/MeshGradient-l2Dz.js)

## React

### Prerequisites
This package requires React 18.0.0 or higher. Make sure you have React installed in your project:

```bash
npm install react
# or
yarn add react
```

### Installation

#### npm
```bash
npm install mesh-gradient
```

#### yarn
```bash
yarn add mesh-gradient
```

**Note:** The required dependencies (`chroma-js` and `twgl.js`) will be automatically installed with the package.
